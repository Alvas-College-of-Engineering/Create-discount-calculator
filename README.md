# 🛍️ Smart Discount Calculator

## 📌 Description
A Java GUI-based application that calculates discounted prices based on user input. Built using Java Swing for a clean and interactive interface.


## 🚀 Features
- Modern Dark-Themed GUI
- Input validation
- Real-time discount calculation
- Clear button functionality
- User-friendly interface

## 🛠️ Technologies Used
- Java
- Swing (GUI)
- OOP Concepts

## ▶️ How to Run

1. Open terminal
2. Navigate to src folder

cd src

3. Compile

javac DiscountApp.java

4. Run

java DiscountApp
